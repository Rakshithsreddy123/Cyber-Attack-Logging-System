# 🛡️ Cyber Attack Logging System

A **Python-based Cyber Attack Logging System** that securely records and stores security events using **Blockchain technology**. The system ensures that attack logs remain **tamper-proof, immutable, and verifiable**, providing a reliable audit trail for cybersecurity analysis.

---

## ✨ Features

- 🔐 Secure logging of cyber attack events
- ⛓️ Blockchain-backed immutable log storage
- 📄 Automatic alert generation
- ⚡ Lightweight and efficient implementation
- 🛡️ Tamper-resistant audit trail

---

## 🎯 Objectives

- Record cyber attack events securely
- Prevent unauthorized modification of logs
- Ensure data integrity using blockchain
- Improve transparency and traceability of security incidents

---

## 🛠️ Tech Stack

- 🐍 Python
- ⛓️ Blockchain
- 📄 JSON
- 📚 Web3.py
- ⚙️ Smart Contract ABI

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/<your-username>/cyber-attack-logging-system.git
```

### Navigate to the Project

```bash
cd cyber-attack-logging-system
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python blockchain.py
```
